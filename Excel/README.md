# Excel Dashboard

This folder contains the Excel reporting and dashboard component of the Retail Sales Analysis project.

## Files

### Retail_sales_analysis.xlsx

Interactive Excel report created using:

* Pivot Tables
* Pivot Charts
* KPI Cards
* Dashboard Layout Design

## Dashboard Pages

### Executive Summary

Provides:

* Total Revenue
* Total Profit
* Total Transactions
* Total Units Sold
* Customer Count

### Regional Analysis

Provides:

* Revenue by Region
* Profit by Region
* Units Sold by Region
* Average Order Value by Region

### Category Analysis

Provides:

* Revenue by Category
* Profit by Category
* Units Sold by Category
* Profit Margin by Category

### Recommendations

Business recommendations based on analysis findings.

## Key Findings

* East region generated the highest revenue and profit.
* Fashion generated the highest profit.
* Electronics generated the highest revenue.
* Groceries had the highest margin percentage but lowest sales volume.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Dashboard Design
