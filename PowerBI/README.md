# Power BI Dashboard

This folder contains the Power BI dashboard developed for the Retail Sales Analysis project.

## Files

### Retail_Sales.pbix

Interactive Power BI dashboard used for business performance analysis.

## Dashboard Pages

### Executive Summary

Provides a high-level overview of:

* Revenue
* Profit
* Transactions
* Customers
* Units Sold
* Monthly Revenue Trend

### Regional Performance

Analyzes:

* Revenue by Region
* Profit by Region
* Units Sold by Region
* Average Order Value by Region

### Product & Category Analysis

Analyzes:

* Revenue by Category
* Profit by Category
* Revenue by Product
* Profit by Product

### Customer Analysis

Analyzes:

* Revenue by Age Group
* Customers by Age Group
* Revenue by Gender
* Profit by Gender
* Customer Join Trend

## Features

* Interactive slicers
* Year filtering
* Gender filtering
* Category filtering
* Dynamic KPI cards
* Cross-filtering visuals

## Key Business Insights

* East region is the strongest performing region.
* Fashion category generated the highest profit.
* Electronics category generated the highest revenue.
* Customers aged 55+ generated the highest revenue.
* Male customers contributed higher revenue and profit.

## Tools Used

* Power BI

* Power Query
* Data Modeling
