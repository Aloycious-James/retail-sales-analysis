# Python Analysis

This folder contains the Python-based analysis of the Retail Sales project.

## Files

### retail_sales.ipynb

Jupyter Notebook containing:

* Data loading and validation
* Exploratory Data Analysis (EDA)
* Distribution analysis
* Correlation analysis
* Profit margin analysis
* Regional performance analysis
* Product and category analysis
* Business insight generation

## Objectives

The Python analysis was performed to:

* Understand data distributions
* Identify relationships between business metrics
* Investigate revenue and profit drivers
* Support findings discovered during SQL analysis

## Key Findings

* Revenue, cost, and profit show strong positive correlation.
* Discount has a negative relationship with profit.
* East region generated the highest revenue and profit.
* Fashion and Electronics categories contributed most revenue and profit.
* Groceries achieved the highest profit margin but generated lower overall profit due to lower sales volume.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook
