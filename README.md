# Retail Sales Analysis

End-to-end retail sales analytics project using PostgreSQL, Python, Excel, and Power BI.

The project analyzes customer behavior, product performance, regional sales trends, profitability, and business opportunities using a retail sales dataset.

## Business Problem

Retail companies generate large amounts of sales data every day.

The objective of this project is to:

- Identify high-performing regions
- Analyze profitable products and categories
- Understand customer purchasing behavior
- Evaluate profit margins
- Discover opportunities to improve revenue and profitability

## Tools Used

- PostgreSQL
- Python
- Pandas
- Matplotlib
- Seaborn
- Microsoft Excel
- Power BI

## Project Workflow

1. Data Collection
2. Database Design using PostgreSQL
3. Data Validation
4. SQL Analysis
5. Python Exploratory Data Analysis
6. Excel Reporting
7. Power BI Dashboard Development
8. Business Recommendations

## Dataset Overview

- Transactions: 5,000
- Customers: 200
- Products: 50
- Stores: 5
- Regions: East, West, North, South

## Key Insights

### Regional Analysis

- East region generated highest revenue and profit.
- South region generated lowest revenue.

### Product Analysis

- And Footwear generated highest profit.
- Set Dairy generated strong profit performance.

### Category Analysis

- Fashion generated highest profit.
- Electronics generated highest revenue.
- Groceries showed highest margin but lowest sales volume.

### Customer Analysis

- Customers aged 55+ generated highest revenue.
- Male customers generated higher revenue than female customers.

## Business Recommendations

- Increase inventory for high-performing products.
- Focus marketing campaigns in low-performing regions.
- Improve customer engagement in South region.
- Review discount strategy for low-performing products.
- Maintain promotional activities for Electronics category.
